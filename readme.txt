Fullscreen Zoom Plugin for Woocommerce

== Description ==

= Overview =

An easy-to-use WooCommerce plugin which makes fullscreen image zooming available. This plugin presents a magnified product images to your customers.

<a href="http://www.barfaraz.com/woofullzoom">Frontend demo</a>

= Features =
* Viewing the details of the product images better
* Fully responsive and 100% customizable
* Easy-to-use & light plugin
* Easy settings panel
* Slider for products & fullscreen thumbnails

= Pro version =
* Vertical slider for large gallery
* Change thumbnails border, preloader and fullscreen colors
* Fade-in images on loading
* More preloaders

<a href="https://codecanyon.net/item/woocommerce-fullscreen-image-zoom/19926273">purchase Pro version</a>

== Installation ==

Log into Admin area of your WordPress website.
Go to "Plugins" —> "Add New".
Find this plugin and click install.
Activate this plugin through the "Plugins" tab.
